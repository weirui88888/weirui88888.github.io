# 介绍

## 用于记录一些工作和学习中遇到的问题，方便回顾和总结。

### 使用方式

- npm run dev 启动本地服务
- 创建 md 文件，配置 SUMMARY.md 目录
- 写完后执行 npm run tree 生成 gitbook 目录
- 替换 README.md 中的目录结构

### 目录结构

```
├── README.md // 介绍页
├── SUMMARY.md // gitbook目录页
├── book.json // 用于配置gitbook
├── note // 文章目录
    ├── Css
    │   └── randomColor.md
    └── Javascript
        ├── firstArray.md
        └── flatArray.md
```
